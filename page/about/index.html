Story

We have been using many technology to improve the efficiency on our daily task, but those product only limits to cellphones, computers, automobiles, and etc... Nevertheless, robotic are sill not implemented in our daily life. The current usage of robot can only be found in factories and the other professional fields. The purpose of this website is to discuss on how to design a robot, and eventually, it should inspire people to develop more robot that could be used in our daily life.

Background

In order to create a better understanding on the general design flow, "Micromouse" is used as an example in this website. Micromosue is an autonomous and self-contained maze solving robot. The aim is to compete with other team by the time used for reaching the maze goal. This project involved programming in ARM environment, analog sensor signal processing, power transmission, system control, motor control, and source power regulation.
